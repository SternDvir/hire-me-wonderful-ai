# Hire Me, Wonderful AI

Automated LinkedIn CTO screening system.

## Documentation

See `/docs/MASTER_BLUEPRINT.md` for complete specifications.

## Setup

1. Install dependencies: `pnpm install`
2. Setup database: `npx prisma migrate dev`
3. Run dev server: `pnpm dev`

## Tech Stack

Next.js 14, TypeScript, Vercel WDK, Prisma, OpenRouter, Tavily
